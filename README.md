Personal research website for [Tenzin Gyaltsen](https://gyaltsentenzin.github.io/).

Built from the [Jon Barron](https://jonbarron.info/) academic website template. Hosted on GitHub Pages.

## Local preview

Open `index.html` in a browser, or run a simple server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Updating content

- **Bio & sections:** edit `index.html`
- **Profile photo:** replace `images/TenzinGyaltsen.png`
- **CV:** replace `data/Tenzin_Gyaltsen_Resume_AIML.pdf`
- **Project thumbnails:** add images under `images/` and update the corresponding `<tr>` rows
